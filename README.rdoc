== README

* Ruby version
2.3.0

* Rails version
4.2.6

* Configuration
Run `bundle install` from root directory of application

* Run application
Run `rails s` from root directory of application, navigate to http://localhost:3000

* How to run the test suite
Run `rspec spec/`

Please feel free to use a different markup language if you do not plan to run
<tt>rake doc:app</tt>.
